Molecule browser (last revision - 15 June 2006)

Molecule browser has three areas:
  • structure diagram is shown on the right;
  • available molecule properties on the left;
  • and finally,at the bottom there is a navigation bar, allowing to sequentially browse molecules in the current data set.

  1. consult How to process current molecule in order to alter/calculate properties of the current molecule, load and save the current data set;
  2. consult How to search in database and Search options in order to load current data set with search results ;
