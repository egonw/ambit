How to create an empty database

  • Go to MySQL server menu
  • Click "Create database". It is necessary to connect to the MySQL server with an user with administrative privileges, such as root user. Enter your password in the corresponding field.


  • Click OK. The following window will  appear, asking for the name of your new database:
  • Type in the name of the database (e.g. "repdose")
  • If  database creation is successfull, you could connect to the new database by going to menu Database/Connect  and typing in the name of the new database in the corresponding field.

  • The new database is empy. To import chemical compounds into database use Database/Database import menu..
