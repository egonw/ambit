How to search in the database. (last revision - 4 June 2006)

All the queries below are applied to compounds in the database.
The results are displayed by default in the Molecule browser, but this could be changed by Result output options.
Currently displayed molecule can be changed by browsing the data set by navigation bar in Molecule browser.

  1. perform a Chemical name search in the database (submenu "Search -> Chemical name");
  2. perform a molecular formula search in the database (submenu ("Search -> Molecular formula");
  3. perform a CAS RN search in the database (submenu "Search -> CAS RN search");
  4. perform a Alias search in the database (submenu "Search -> SMILES");
  5. perform a SMILES search in the database (submenu "Search -> SMILES");
  6. Search AQUIRE database;  (submenu "Search -> Search AQUIRE");
  7. Retrieve named subset from the database;  (submenu "Search -> Retrieve dataset");
  8. define structure,descriptor,distance-based and experimental data criteria and perform searches in the database database, while observing application's response time statistics, shown in the status bar;  (submenu "Search -> Structure search");
    • Search by structure. Specify query and click Exact search, Substructure search or Similarity search button.

    • Search by descriptors. Specify query and click Search button.

    • Search by experimental data. Specify query and click Search button.

  9. browse the result set and perform actions explained in How to process current molecule compounds;
