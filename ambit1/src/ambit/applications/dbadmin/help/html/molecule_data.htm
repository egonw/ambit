How to observe AQUIRE data and metabolites (last revision - 17 June 2006)

These tabs display information about the current molecule in Molecule browser.

AQUIRE

This tab displays AQUIRE data for the current compound (if available). Click on AQUIRE in Molecule menu to retrieve AQUIRE data for the current molecule.

Metabolites

Only a limited number of hydrolysis reactions are implemented, in order to illustrate the concept. Click on Metabolites in Molecule menu to generate metabolites of the current molecule.
