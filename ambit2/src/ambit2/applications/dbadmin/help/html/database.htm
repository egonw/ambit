AMBITDatabase HELP

How to calculate chemical structures into database. (last revision - 14 June 2007)


Connect : connects to a user specified database. User and password have to be provided
Status: displays current status (Connected / not connected)
Statistics: number of records in various tables.


Connect

If  there is already a connection to the database (see Status), the system will display a warning :


In order to connect to another database, answer Yes, otherwise to keep the same connection, click No.  If the answer was Yes, the following dialog will appear,
asking for connection details:

  • user  - Enter an username, as created by MySQL server/Create User menu or default root user 
  • password - Enter password as created by MySQL server/Create User, or root password, as assigned when installing MySQL server. The default installation comes with an empty password.
  • host - Enter the name or IP address of the machine, where the database runs. By default, it is the same computer AmbitDatabaseTools is running on (use "localhost" or "127.0.0.1" to denote the local computer).
  • port - Enter the port number MySQL server is running on. The default installation of AmbitDatabaseTools comes with a MySQL server, running on port 33060. Don't change this setting, unless you are experienced user. 
  • database - the name of the database, as created by MySQL server/Create database, or default ambit .
