How to perform batch processing of large files. (last revision - 17 June 2006)

Batch processing of an user specified files. Several options, explained below are available.
The results are saved in a file, also specified by the user. Does not affect molecules in Molecule browser.

1.Select input file
2.Select output file
3.Select processing options
4.Click start button to start processing
5.Pause button can be clicked at any time, in order to temporarily stop processing
