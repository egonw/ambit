AMBITDatabase HELP

How to import chemical structures into database. (last revision - 11 July 2007)

Imports structures and other data into the database you are currently connected to (see Database/Status).  

The imported data can be searched by Search option or retrieved by Search/Retrieve dataset option.

Import compounds and descriptiors from SDF, CML, CSV, TXT files.

Please pay attention CSV , TXT and XLS  formats should have headers (first row with column names). The system will ask which column should be used for what.
