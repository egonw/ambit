AMBITDatabase HELP

(last revision - 4 June 2006)

The complete AMBITDatabase documentation is being developed and will be located in the "doc" application subfolder when available.

Meanwhile we provide here only a brief outline of common AMBITDatabase usage:

  1. launch the application from the Start Menu "Start -> All Programs -> CEFIC-LRI -> AMBITDatabase-vX.YZ -> AMBITDatabase-vX.YZ";
  2. consult User interface;
  3. consult How to process current molecule in order to alter/calculate properties of the current molecule, load and save the current data set;
  4. consult How to search in database and Result destination options in order to load current data set with search results ;
