Implementation of batch processing.

Batch processing is a central concept in AMBIT. Almost all processing is done via {@link ambit.io.batch.IBatch} implementations. See also {@link ambit.ui.actions.BatchAction} - A batch action (can be added in a menu or invoked from a button) {@link ambit.ui.batch.BatchProcessingDialog} - A dialog displaying various properties of a {@link ambit.io.batch.IBatch}
